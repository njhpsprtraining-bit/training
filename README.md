# NJHPS Industrial Training Management System

**नाथपा झाकड़ी हाइड्रो पावर स्टेशन — औद्योगिक प्रशिक्षण प्रबंधन प्रणाली**

## Features
- 📊 Dashboard with live dept activity
- 👥 Trainee registry (107 pre-loaded from Google Sheet)
- 📋 Group orders with staggered dept rotation (no dept clash)
- 📄 Hindi order generation with SJVN logo
- 🏭 Department live view
- 🎓 Certificate & dispatch tracking
- 📈 Reports: Status, Dept-wise, Institution-wise, Annual
- 📱 Mobile responsive with bottom navigation
- 🔒 Admin password protected

## Usage
Open `NJHPS_Training_System.html` in any browser.

Default password: `njhps2024`

## Tech
Single-file HTML app with vanilla JS. Data stored in browser localStorage. Google Sheet sync via CSV export.
